\# NOTES.md

The best-performing configuration used a 4-layer Transformer with an embedding dimension of 184, 4 attention heads, Rotary Position Embeddings (RoPE), and a SwiGLU feed-forward network. This configuration achieved a validation BPB of \*\*2.0876\*\* with \*\*1,724,080\*\* parameters, remaining comfortably under the 2M parameter limit. RoPE produced the largest single improvement by replacing learned positional embeddings with relative positional information. Replacing the standard GELU MLP with SwiGLU further improved representation quality. Increasing model width utilized the available parameter budget effectively and produced an additional improvement. Experiments with weight tying, GPT-2 style initialization, RMSNorm, QK-Norm, optimizer modifications, and logit soft-capping did not improve validation BPB and were therefore discarded. The final model retains only experimentally validated improvements while keeping the architecture simple, reproducible, and fully compliant with the assignment constraints.
